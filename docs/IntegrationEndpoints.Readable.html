SCANPAN API-DW Cloud

Integration Endpoints

Læsevenlig oversigt over de tre operationelle OData-sider i cloud-projektet: AuningStockDW, PerfionItemsDW og PerfionPricesDW.

Formål

Dokumentet beskriver funktion, teknik og drift for de tre centrale OData-endpoints.

Status

Tilpasset cloud-repoets aktuelle kode og publiceringslogik pr. 2026-04-16.

Samlet overblik

Service Formål Datakilde Mønster
AuningStockDW Snapshot af aktuelt lager til eCommerce Item med snapshotfelter Job Queue beregner, OData eksponerer
PerfionItemsDW Kurateret varemaster til Perfion-relateret eksport Item plus relaterede tabeller Direkte OData med enrichment
PerfionPricesDW Kurateret prisfeed for gyldige priser Price List Line og Campaign Temporary dataset bygget ved kald
Perfion custom API-siderne findes stadig i appen, men OData-siderne er de operative publiceringer i denne løsning.

1. AUNING Stock

Scope

  • Lokation AUNING
  • Kun inventory items
  • INTERN|EKSTERN|BRUND

Felter

  • itemNo
  • description
  • auningStockOnHand
  • auningStockAvailable
  • auningStockUpdatedAt

Drift

  • Job Queue-kørt snapshot
  • Vises også i Item Card FactBox
  • Heltal, aldrig negativt

AUNING Stock On Hand beregnes fra lagerbeholdning pr. lokation og variant. AUNING Stock Available beregnes via standard warehouse-availability logik og kan reduceres med procent. 

GenProdPostingGroupFilter=INTERN|EKSTERN|BRUND
GenProdPostingGroupFilter=INTERN|EKSTERN|BRUND;AvailableReductionPct=10

2. Perfion Items

PerfionItemsDW eksporterer en kurateret item-master med type, klassifikation, oversættelser, barcodes og UoM-data.

  • Filter på produktbogføringsgruppe, ABCD-kategori, usage og produktgruppe
  • Oversatte beskrivelser i flere sprog
  • Barcodes for STK, INNER og MASTER
  • Mål og vægt pr. UoM

3. Perfion Prices

PerfionPricesDW bygger et midlertidigt prisdatasæt og vælger én foretrukken række pr. logisk kombination.

  1. Kun aktive prislinjer gyldige i dag
  2. Seneste Starting Date vinder
  3. Laveste pris vinder ved samme startdato
Cloud-repoet har i øjeblikket kampagnematch slået ud i koden. Derfor returnerer campaignPrice = 0 og tomt campaignId, selv om felterne findes i output.

Publicering og rettigheder

DW WS Registrar publicerer de tre services som: AuningStockDW, PerfionItemsDW og PerfionPricesDW.

Rettighederne håndteres via separate permission sets for AUNING stock og Perfion.

Kodeeksempel

local procedure NormalizeAvailableQuantity(AvailableQuantity: Decimal; AvailableReductionPct: Decimal): Decimal
var
    ReducedAvailableQuantity: Decimal;
begin
    ReducedAvailableQuantity := AvailableQuantity;
    if AvailableReductionPct > 0 then
        ReducedAvailableQuantity := ReducedAvailableQuantity * (100 - AvailableReductionPct) / 100;

    exit(NormalizeQuantity(ReducedAvailableQuantity));
end;

Forklaring: Disponibel beholdning reduceres kun, hvis parameteren er større end 0. Derefter normaliseres resultatet til heltal og minimum 0.
